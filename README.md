# 海外公司开源运维方案

本仓库收录不同人分享的海外公司注册、报税、合规等相关信息。欢迎大家提交PR，主要面向独立开发者人群。
这个仓库也是个 Obsidian Vault，你可以 git clone 到本地后用 Obsidian 打开。
